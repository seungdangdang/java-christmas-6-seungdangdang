### 주문 메뉴 / 개수 입력 시

- [x] 메뉴판에 있는 메뉴인가?
- [ ] 음료만 주문했는가?
- [ ] 총 주문 금액이 10000원 이상일 때 할인 이벤트를 적용했는가?
- [X] 메뉴의 개수가 최소 1이상인가?
- [ ] 중복 메뉴가 없는가?
- [ ] 메뉴를 20개 이하로 주문했는가?
- [x] 메뉴의 형식이 ([메뉴]+”-”+[수량])이 맞는가?
